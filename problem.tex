\section{Problem Definition}
Suppose there are $N$ agents interacting in an environment. 
The goal of this paper was to maximize the discounted cumulative reward the system obtained from the external environment.
This was calculated as:
\begin{equation}
G = \sum_{i=1}^N \left[ \sum_{t=0}^T \gamma^t R_{it}^\mathrm{ex} \right],
\end{equation}
where $R_{it}^\mathrm{ex}$ is an external reward which $i$-th agent obtains at $t$, and $\gamma \in [0, 1]$ is the discount rate and $T$ is the terminal time.
