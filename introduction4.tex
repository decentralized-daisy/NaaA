\section{Introduction}
Deep reinforcement learning (DRL) succeeds in many areas such as TV games  and Board games.
Such success is supported by generalization capability of neural network, which maximizes technique.

ここ数年の DRL の成功は目覚ましいものがあり、テレビゲームや囲碁で次々に人間を破っている。
こうした成功は、ニューラルネットワークによる状態表現を抽象化し、有用な表現を獲得する能力によって下支えされている。
そのため、DRL はいわゆる AI のブレインとして、自動運転、金融取引、農業など各種産業への応用が注目されている。

一方、DRL を実世界への応用するときに問題になるのが、どのように環境の観測を行うかである。
テレビゲームやボードゲームなどの人工の環境と異なり、
実環境はある時点における情報を一人のエージェントによって完全に観測することはできない。
こうした問題は、次の二つのアプローチによって解決することができる。
一つは、ニューラルネットワークによる環境の遷移についての予測を行うこと。
これは、RNN などリカレントなモデルを搭載した単一のエージェントが環境内を積極的に探索することにより、できる限り多くの情報を集めることにより実現される。
もう一つは、マルチエージェント環境を用意し、コミュニケ―ションによって環境を間接的に観測するということである。
前者は、環境が広くなった場合に、十分なデータを得るために時間の制約が存在する。
後者のアプローチでは、エージェントの数を増やせばよく、インターネットなどの通信回線を利用することによって世界規模・宇宙規模でのスケールが可能である。
そこで、本研究では、後者のアプローチを取る。
後者のアプローチには、CommNet などの既存研究が存在する。

従来の研究において解決されていなかった課題は、
各エージェントが利己的であると仮定した場合の報酬の配分方法である。
ウェブのような環境で考えた場合、エージェントは別々の人間や会社が作る可能性があり、
必ずしも共通の目的を持っているわけではない。
最適な報酬配分が実現できなければ、エージェントを作成するインセンティブも発生しない。
これには、いくつかの論点がある。ひとつは「いくらの報酬額が妥当であるのか」、
もう一つは「いかにエージェントに報酬の支払いを義務づけるか」である。
前者は信頼度割り当て問題として、マルチエージェント強化学習(MARL)でよく知られた問題であり、
後者は社会的ジレンマの解決として知られている。
近年、ブロックチェーン技術の発展により、ビットコインやイーサリアムなどの仮想通貨を用いてエージェントへのインセンティブの配分が可能になっている。
報酬の原資は、環境を観測することで利益を得ることができるアクチュエーター側が得た予算から支払われる。

本研究では、この問題を経済学を応用した Neuron as an Agent (NaaA) というゲームデザインを独自に設ける。
本来、経済学には複数の個が存在する社会において、最適な資源配分を実現するというモチベーションがある。
NaaA 上では、各エージェントが表現と報酬の取引を行うものであり、
次の二つの重要な概念によって構成されている。
一つは counterfactual return であり、もう一つは envy-free auction である。
Counterfactual return は counterfactual reward \cite{} にヒントを得ている。
Envy-free auction は経済学の一つであるゲーム理論における、メカニズムデザインを応用したものである。
オークションは経済学において、商品を高く評価している人に、商品がいきわたりやすくするゲームデザインである。
メカニズムデザインは逆ゲーム理論とも呼ばれ、最適なゲーム設計を考える分野である。

エージェントにはいくつかのタイプがある。
一つは各エージェントから得られた情報を統合し行動に結びつけるアクチュエーターであり、
環境に対して影響を与えることで直接報酬を受け取る。
二つ目は環境を直接観測するエージェント、三つめは他のエージェントから情報を受け取り、変換して出力するエージェントである。
これらは、ニューラルネットワークにおける出力層、入力層、中間層に対応する。
究極的に、各エージェントに含まれるニューロンの数は一つでもよく、この場合は
ニューロン自体をエージェントとみなしたモデルになる。
NaaA は、このモデルを想定する。
